PREREQUISITES:
For systems running Windows 7:
- TPM 1.2 must be activated in BIOS (confirm the TPM state in the BIOS).

For systems running Windows 8.x or Windows 10:
- The OS will automatically take ownership of TPM.
- Windows 8.x and Windows 10 require GPT partition style when using TPM 2.0. The
BIOS setting for boot mode should be set to native UEFI (recommended) or UEFI
with CSM.

- Systems running Windows 7 (64 bit), Windows 8.x, or Windows 10 will work with
MBR or GPT partition style when using TPM 1.2.

HOW TO USE:
1. Download the file by clicking the Download or Obtain Software button and
saving the file to a folder on your hard drive (make a note of the folder where
the downloaded file is saved). NOTE: The default directory location is
C:\SWSETUP\SP87492.

2. Double-click the downloaded file and follow the on-screen instructions.
